
# ToDo-without-redux
Task list application using React without Redux. The backend uses MongoDB and Express.


# Instalation

1. Install MongoDB and start backend:
- sudo apt install mongodb-server-core
- cd backend
- mkdir mongo-db
- mongod --dbpath=./mongo-db
- npm install
- npm run dev

2. Start front end:
- cd front end
- npm install
- npm run dev
